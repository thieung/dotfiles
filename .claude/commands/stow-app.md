# Stow a Specific Application

Install configuration for a specific application.

Usage:
```bash
./install.sh stow-app <app-name>
```

Available apps: nvim, fish, tmux, git, kitty, alacritty, yabai, skhd

Example:
```bash
./install.sh stow-app nvim
```
